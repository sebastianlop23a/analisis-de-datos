🚀 CÓMO USAR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Asegúrate que el servidor Spring Boot está corriendo:
   $ mvn spring-boot:run
   O con Docker:
   $ docker-compose up

2. Abre tu navegador en:
   http://localhost:8080

3. La interfaz cargará automáticamente con el dashboard

4. Comienza creando una máquina:
   - Sección "Máquinas" → "Crear Nueva Máquina"

5. Crea un ensayo:
   - Sección "Ensayos" → "Crear Nuevo Ensayo"

6. Registra datos:
   - Opción A: Manual → "Registrar Datos en Ensayo"
   - Opción B: Archivo → "Cargar Datos desde Archivo"

7. Analiza los datos:
   - Sección "Análisis" → Selecciona ensayo
   - Ver gráficos y estadísticas automáticamente

8. Genera reportes:
   - Sección "Reportes" → "Generar Reporte"
   - Descarga en tu formato preferido

   const API_CONFIG = {

   BASE_URL: 'http://tu-dominio.com/api'
