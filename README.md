# Sistema de Análisis de Datos

Aplicación desarrollada en Java (Spring Boot) para el procesamiento, análisis y generación de resultados a partir de archivos estructurados como Excel y PDF.

## Problemática

En entornos empresariales, el análisis de datos provenientes de archivos como Excel y PDF suele realizarse de forma manual, lo que genera retrasos en la obtención de resultados, alta probabilidad de errores humanos y procesos repetitivos poco eficientes.

## Objetivo

Automatizar el procesamiento y análisis de datos mediante una aplicación que permita leer, transformar y generar resultados de manera estructurada y confiable.

## Funcionalidades

- Lectura de archivos Excel  
- Procesamiento de archivos PDF  
- Transformación y limpieza de datos  
- Generación de resultados estructurados  
- Automatización de procesos manuales
